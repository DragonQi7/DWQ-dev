Auxiliary Star Finder

Yellow / Black Belt Client-side JS Logic ported from find_auxiliary_star.py
Inputs Standalone HTML
Uses the same date parsing order as the Python script: YYYY-MM-DD, MM/DD/YYYY, DD/MM/YYYY, month-name formats, or YYYYMMDD.
Range mode walks forward day by day from the selected start date and applies the same month-animal and auxiliary-star rules as the Python file.
Range mode shows one row per day. Star Filter and Belt Filter are optional and behave like the Python CLI.
This page runs fully in the browser. It does not call Python directly. The day-pillar anchor, Jieqi month boundaries, auxiliary-star lookup table, belt classification, and star definitions are all embedded from the current Python logic.
